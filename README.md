🔐 Project Lucia

Project Lucia is a next-generation AI-powered cybersecurity web application designed to create strong, secure, and unpredictable passwords using user-defined inputs. It combines intelligent pattern detection, real-time analysis, and a modern cyberpunk interface to make online security simple and effective for everyone.

---

## 🚀 Features

### 🔑 Advanced Password Generator
- Generate strong passwords using words, numbers, and symbols
- Multiple modes: Normal, Passphrase, Ultra Secure, System
- Cryptographically secure randomness (CSPRNG)
- Character transformations
- Adjustable length and batch generation
- Base32 / Base64 encoding options
- Estimated crack time display (e.g., years, centuries)

---

### 🧠 Smart Security Analyzer
- Real-time password strength analysis
- Detects weak patterns (e.g., sequential, repeated, common passwords)
- Provides suggestions to improve security
- Entropy-based evaluation

---

### 📧 Breach Detection (API Integrated)
- Check if your **email** has appeared in known data breaches
- Detect commonly leaked or weak password patterns
- Uses secure API integration (no sensitive data stored)
- Helps users avoid compromised credentials

---

### 🔐 Private Password Vault
- Store passwords securely in your browser (localStorage)
- No cloud sync — fully client-side
- Add labels for easy organization
- Export saved passwords as PDF for offline backup

---

### 📁 File Hash Checker
- Generate hashes: SHA-256, MD5, SHA-1
- Verify file integrity
- Detect file tampering

---

### 🤖 Lucia AI Assistant
- Interactive virtual assistant
- Guides users in generating stronger passwords
- Provides contextual feedback
- Enhances overall user experience

---
